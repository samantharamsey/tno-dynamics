# Numerical Methods

This document will describe the numerical algorithms used in the project.

## Planned Methods

- numerical integration;
- event detection;
- variational-equation propagation;
- differential correction;
- pseudo-arclength continuation;
- periodicity validation;
- linear stability analysis;
- Poincaré maps;
- resonance diagnostics.

## Validation Requirements

Each numerical method will be accompanied by an appropriate analytical,
physical, or numerical validation.
