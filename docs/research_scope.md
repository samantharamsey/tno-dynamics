# Research Scope

## Scientific Objective

This document will define the scientific goals and boundaries of the research project.

## Initial Numerical Foundation

The initial implementation will include:

1. Circular restricted three-body problem dynamics.
2. Variational equations and state transition matrix propagation.
3. Event-based numerical integration.
4. Differential correction for periodic orbits.
5. Pseudo-arclength continuation of periodic-orbit families.
6. Periodic-orbit stability analysis.
7. Poincaré maps and resonance diagnostics.

## Longer-Term Research Direction

The numerical foundation will support later investigations of resonant
dynamics, chaotic transport, resonance sticking, and trans-Neptunian
object dynamics.

## Current Status

The project is being initialized from scratch. No scientific implementation
has yet been accepted as validated.
