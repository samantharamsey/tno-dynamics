# Mathematical Conventions

This document will define all mathematical conventions used by the project.

## Items to Define

- dimensional and nondimensional quantities;
- reference frames;
- primary-body locations;
- CR3BP mass parameter;
- state-vector ordering;
- state transition matrix convention;
- matrix-flattening convention;
- Jacobi constant convention;
- integration-time convention;
- event definitions;
- periodic-orbit symmetry assumptions;
- continuation variables and parameters.

## Current Status

These conventions have not yet been finalized. Scientific code should not
silently assume conventions that are absent from this document.
